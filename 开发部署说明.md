## 🚀 部署步骤（非常简单）

### 第一步：导入项目到微信开发者工具

1. 打开微信开发者工具
2. 点击「导入项目」
3. 项目目录选择：`copywriter-miniprogram`
4. AppID 已经配置好：`wxfd03c690687b9ce6`
5. 点击「导入」

### 第二步：配置API Key

1. 打开 `miniprogram/config.js` 文件
2. 在文件开头找到配置区域：
   ```javascript
   const API_CONFIG = {
     // 在这里填写你的API Key
     doubaoApiKey: '你的豆包API Key',
     // 或者其他模型的API Key
     // 当前选择使用的AI模型
     activeModel: 'doubao',
     // 每日限制使用次数
     DAILY_LIMIT: 10
   }
   ```
3. 填入你自己的API Key，选择你要使用的模型

**推荐使用豆包**：价格便宜，国内访问速度快，价格约每千token几分钱。

### 第三步：配置服务器域名

1. 打开 [微信公众平台](https://mp.weixin.qq.com/) → 登录你的小程序 → 开发 → 开发设置
2. 找到「服务器域名」→ 修改 → 添加request合法域名
3. 根据你使用的模型添加对应的域名：

   - **豆包（字节跳动）：** `https://aquasearch.ai.bytedance.net`
   - **OpenAI：** `https://api.openai.com`
   - **通义千问（阿里）：** `https://dashscope.aliyuncs.com`
   - **文心一言（百度）：** `https://aip.baidubce.com`

4. 保存修改

### 第四步：编译运行

1. 回到微信开发者工具
2. 点击「编译」按钮
3. 直接运行测试！

**不需要**：
- ❌ 不需要开通云开发
- ❌ 不需要创建云数据库
- ❌ 不需要部署云函数
- ❌ 不需要后端服务器

就是这么简单！

## 📝 配置每日使用次数限制

打开 `miniprogram/config.js`，修改 `DAILY_LIMIT` 即可：

```javascript
const API_CONFIG = {
  // ...其他配置...
  DAILY_LIMIT: 10  // 修改这里，设置你想要的每日次数
}
```

次数统计是基于本地存储，按日期区分，每天自动重置。

## ⚠️ 注意事项

1. **API Key安全：** API Key存储在前端代码中，如果是公开项目，可能会被他人看到。仅供个人开发者小型项目使用。
2. **网络请求：** 由于是前端直接调用API，需要在微信公众平台配置服务器域名，不然会被拦截。
3. **每日限制：** 由于是本地存储，用户清除小程序缓存后次数会重置。这是MVP简化方案，完全可用。
4. **支持多种模型：** 可以随时在 `config.js` 中切换模型，不需要修改其他代码。

## 🔧 常见问题

**Q: 生成失败提示"请先配置API Key"**
A: 去 `miniprogram/config.js` 配置你的API Key，然后重新编译即可。

**Q: 提示"今日次数已用完"**
A: 每天限制使用次数，次日会自动重置。如需修改次数限制，修改 `config.js` 中的 `DAILY_LIMIT`。

**Q: 提示"网络请求失败"或者"API调用失败"**
A: 检查：
1. 是否已在微信公众平台配置了对应的服务器域名
2. API Key是否正确填写
3. API账号是否有余额

**Q: 复制功能不工作**
A: 复制需要微信小程序基础库2.0+支持，开发者工具默认都支持，如果真机不行请升级基础库版本。

**Q: 提示"内容包含敏感信息"**
A: 输入或生成结果中包含敏感词，请修改后重试。如需调整敏感词列表，可以编辑 `app.js` 中的 `sensitiveWords` 数组。
