
# PRD: 朋友圈文案润色小程序 (MVP v1.1)

## 产品定位

帮助用户快速润色原始文案，生成适合朋友圈的优美文案，降低用户表达门槛，提升发朋友圈体验。

## 技术架构

- **前端:** 微信原生小程序
- **后端:** 微信云开发（云函数 + 云数据库）
- **不需要独立服务器**
- **AI:** 调用第三方大模型API（豆包/OpenAI/通义千问/文心一言）

## 交互流程

```
用户打开小程序
  ↓
首页输入原始文案
  ↓
选择润色风格
  ↓
点击"开始润色"
  ↓
检查今日配额，如果还有次数 → 调用AI生成
  ↓
展示3个润色结果
  ↓
用户点击喜欢的结果 → 复制到剪贴板
  ↓
提示用户去朋友圈粘贴发表
  ↓
生成记录保存到历史记录
```

## 设计规范

- 主色调：#1677ff（微信蓝）
- 背景：#f5f5f5
- 卡片设计，圆角阴影
- 字体大小适配移动端
- 简洁清晰的交互

## 验收标准

1. 能够正常输入文案，字数统计正确
2. 能够选择风格，选中状态正确
3. 点击生成能够正确调用AI，返回结果
4. 点击结果能够正确复制到剪贴板
5. 次数统计正确，每日重置
6. 历史记录能够正常显示
7. 敏感词能够正确过滤
8. 在iOS和安卓都能正常运行
9. 代码结构清晰，有注释
10. 留出API Key配置位置
